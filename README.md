# 贾志超人像摄影作品集 - 横屏放映版

静态 GitHub Pages 网站，面向电脑 16:9 横屏展示和面试现场讲解。

## 本地预览

```powershell
py -m http.server 5182 --directory D:\CODEX5\jzc-photo-screen
```

打开 `http://127.0.0.1:5182`。
